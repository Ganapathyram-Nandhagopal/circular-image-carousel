### Change Images

Update the image array inside `script.js`:

```javascript
const imgUrls = [
  "image1.jpg",
```

### Adjust Card Size

Modify the CSS variables:

```css
:root {
  --card-w: 200px;
  --card-h: 300px;
  --radius: 750px;
}
```

| Variable | Description              |
| -------- | ------------------------ |
| --card-w | Card Width               |
| --card-h | Card Height              |
| --radius | Carousel Rotation Radius |
